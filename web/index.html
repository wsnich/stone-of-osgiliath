Connecting…
Follow these steps in order. Each one unlocks the next feature. You can always change settings later via the ⚙ gear icon.
1 Start the app & open the browser
Run start.bat (Windows) or python main.py. A Chromium window will open — leave it running in the background.
2 Log in to Discord in the browser window
The Chromium window will show Discord's login page. Sign in with your email, password, and 2FA. You only need to do this once — the session is saved.
3 Add Discord channels to monitor
Open ⚙ Settings → Discord and paste your email/password so the app can reconnect automatically. Then scroll down to the Discord feed panel and click Options → add a channel URL.

To get a channel URL: right-click any channel in Discord → Copy Link. Paste the full URL.
4 Add products to the TCGPlayer tab
Click the TCGPlayer tab, then + Add TCGPlayer. Paste a TCGPlayer product URL and give it a name. The app will start tracking its price, listings, and sold data.

For foil singles, add a ManaPool URL in the product form to get live ManaPool pricing alongside TCGPlayer.
5 Set up Discord DM notifications (optional)
Create a bot at discord.com/developers → Applications → Bot → Reset Token. Paste the token in Settings → Discord → Bot Token and your User ID in the field below it. The app will DM you when a deal is detected.
6 Start the monitor
Click ▶ Start in the header. The status dot will turn green and the app will begin checking prices and watching Discord channels. That's it.
Tip: The Deals tab auto-aggregates Discord alerts matching your keywords. The Intelligence tab builds a retailer map automatically from those same messages — no setup needed.
Sealed / Other Uses the global Check Interval from Settings (default 5 min). Each product is offset so checks spread out instead of piling up. Singles Fixed at 6 hours regardless of the global interval — singles change slowly so frequent checks are wasteful. Graded / Comics Uses the Graded Interval from Settings (default 7 days). Per-product override Any product can have its own interval set when you add or edit it. This overrides the global and type-based intervals.
Jitter — each interval gets a small random offset (±20%) so checks don't all fire at once. A 5-min interval with 20% jitter fires somewhere between 4–6 min.
Restart behaviour — on startup, each product's next check is calculated from its last-checked time plus the interval, so recently-checked products aren't immediately re-checked.
Countdown pill — the "next check" timer in the header shows when the earliest queued check will fire across all active products.
Settings
Discord ▼
The app logs into Discord as you in a hidden browser window and watches deal channels in real-time. Your credentials are only used to open that browser session — they're never sent anywhere else. The Bot Token is separate and only needed if you want push notifications sent to your phone via DM.
Logs into Discord web in a headless browser. Looks like a normal browser session. Requires restart.
Monitor & Schedule ▶
Controls how often sealed products are price-checked (singles are fixed at 6 hrs, graded at the interval below). Jitter adds a small random offset so all products don't fire at the same time. Active hours pauses all checks outside the window you set — useful if you don't want the browser running overnight.
Browser ▶
The app uses a real Chromium browser (via Patchright) to scrape TCGPlayer, eBay, and Discord — it looks like a normal user visit. Leave these on defaults unless you're having detection or timeout issues. Headless mode keeps the browser window hidden; disable it if you need to watch what the browser is doing to debug a problem.
One proxy per line. Each product check picks a random proxy. Failed proxies are skipped until all have been tried.
Reddit ▶
Polls the new feed of any subreddits you list and surfaces posts in the Reddit panel at the top of the page. No Reddit account needed — it uses the public JSON API. Good sources: sealedmtgdeals, mtgfinance, pokemontcg.
Research Agent ▶
An AI agent (Claude) that periodically reviews your price history, eBay sold comps, and Discord deal data to surface trends and anomalies. Results appear in the Research tab. Requires an Anthropic API key — runs cost a few cents each. Set the interval to 168 hours (weekly) to keep costs minimal.
Data ▶
All app data (price history database, saved state, deals, portfolio) is stored in a single folder. By default this is the project root next to main.py. Set a custom path if you want to keep data on a separate drive or shared folder. Changes require a restart.

Welcome! Let's get you set up.

This tool monitors Discord channels for deals on trading card games and collectibles. To get started, enter your Discord login so the app can monitor deal channels in real-time.

Enter your Discord email and password. The app logs into Discord web in a headless browser to monitor deal channels in real-time.

In Discord, enable Developer Mode (Settings → Advanced). Right-click a deal channel → Copy Channel ID.

Create a bot at Discord Developer Portal, get its token, and enter your User ID to receive deal alerts on your phone.

All settings can be changed later via the ⚙ gear icon in the header.
Your tokens are stored locally in config.json and never shared.
